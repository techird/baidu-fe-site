GMU

Global Mobile UI

GMU是基于Zepto并且面向移动端的UI组件库，价值在于为移动端快速开发提供稳定、丰富的UI组件。

产品线应用

目前，GMU在各产品线有着非常广泛的应用，包括但不限于新闻、图片、知道、文库、百科、地图、音乐、视频、旅游等一系列产品线，大大提高了广大一线前端工程师们的开发效率。


此外，我们与产品的工程师们持续保持着沟通，积极跟进反馈意见，根据需求及时开发新组件，不断丰富GMU。

SUGGESTION组件

搜索建议组件，算得上是使用频率最高的组件之一，拥有简洁的接口和调用方式。

SLIDER组件

能够便利的提供图片轮播功能，更有灵活滑动、无缝切换、动态载入等实用功能。

SUGGESTION组件

非常简便的实现APP导航功能，丰富的事件派发，可自由配置的滑动导航、FIX导航等。
